# @jieonist/vision-camera-ocr-scanner

On-device OCR structured scanner (card, business card, receipt, MRZ) for React Native — built on VisionCamera + Nitro

## Installation


```sh
npm install @jieonist/vision-camera-ocr-scanner react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { multiply } from '@jieonist/vision-camera-ocr-scanner';

// ...

const result = multiply(3, 7);
```
